# Envir Galaxy Technology

Modern React-based agency website inspired by contemporary SaaS and technology company designs.

## Features

- Dark UI
- Fluid Interactive Background
- Responsive Design
- Framer Motion Animations
- Mobile Navigation
- Testimonial Slider
- Portfolio Showcase

## Tech Stack

- React
- Vite
- CSS3
- Framer Motion
- React Icons
- Three.js
- React Three Fiber

## Installation

npm install
npm run dev

## Build

npm run build

## Project Structure

src/
├── components/
