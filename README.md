# Envir Galaxy Technology

A modern technology company website built with React and Vite, featuring a dark-themed user interface, smooth animations, responsive design, interactive fluid background effects, and a clean user experience.

## Live Demo

https://envir-galaxy-technology-clone.vercel.app/

## Features

* Modern dark-themed UI
* Responsive design for desktop, tablet, and mobile devices
* Interactive fluid background effects
* Sticky navigation bar
* Smooth scrolling navigation
* Animated hero section
* Services showcase section
* Why Choose Us section
* Portfolio showcase
* Animated testimonial slider
* Contact section
* Professional footer
* Framer Motion animations
* React component-based architecture

## Tech Stack

### Frontend

* React
* Vite
* JavaScript (ES6+)
* CSS3

### Libraries

* Framer Motion
* React Icons

* React Three Fiber
* React Postprocessing
* Fluid Distortion Effects

## Installation

Clone the repository:

```bash
git clone https://github.com/yashprajapati18705-ctrl/Envir-Galaxy-Technology.git
```

Navigate to the project directory:

```bash
cd Envir-Galaxy-Technology
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Production Build

Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

This project is deployed using Vercel.

To deploy your own version:

1. Fork or clone the repository
2. Push it to GitHub
3. Import the repository into Vercel
4. Deploy with default Vite settings
